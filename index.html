// Installation

Zero to running
in minutes.

One command on Linux (Debian, Ubuntu, Fedora, Arch) and macOS. Windows is supported through WSL2. The installer is non-destructive — it detects existing Docker and reuses it.

01
Run the installer

A single curl command detects your OS, verifies Docker and pulls the QueAI core image. Works on any platform.

curl -fsSL https://raw.githubusercontent.com/queai-project/QueAI/main/install.sh | bash
02
Open the dashboard

Navigate to localhost:8080 in your browser. QueAI's module marketplace is ready.

03
Install your modules

Browse the catalog, click Install. Each module pulls its Docker image, exposes its own UI and API, and starts isolated on your machine.

04
Start using AI

Chat with documents, transcribe audio, extract text from images. All orchestrated by QueAI, all under your control.

// Product Principles

What drives us.

🔒
Local-First & Private

Your data never leaves your machine. Zero telemetry, zero mandatory external calls. Air-gappable by design.

🐳
Docker-Native

Each module is an isolated Docker container with its own dependencies. Install, update or remove without polluting the host system.

🆓
Free Core, Always

The QueAI orchestration core is fully open source and free. No freemium gates, no hidden limits in the base platform.

Extreme Modularity

Every capability is a decoupled plugin. Activate what you need, remove what you don't. No bloat, no assumptions.
